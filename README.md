# CLI control player
Use AppleScript to Control player play, pause, next, previou track.

## How to use
Run in terminal
```
osascript itunes_play.scpt
# or
./play.sh itunes play
```

Run with other shortcut tools for extend functions, such as *bettertouchtool*
